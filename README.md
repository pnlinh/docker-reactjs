# How to start

### Install [Makefile](https://makefiletutorial.com/) on your OS (recommend)

### To install project
```shell
make install
```

### To start project
```shell
make start
```

### To start project
```shell
make stop
```

### To execute container
```shell
make shell
```

### To debug container log
```shell
make logs
```
